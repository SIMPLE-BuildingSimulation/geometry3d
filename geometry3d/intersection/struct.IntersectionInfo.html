Expand description

Contains more detailed information about the what is happening at the surface in the intersection point

Fields§

§p: Point3D

The point of intersection

§normal: Vector3D

The normal Vector3D at the interaction. Must have the value of (dpdu x dpdv).normalize() … use ShadingInfo::new(..) and this is done automatically

§side: SurfaceSide

The side in which the ray intersected

§dpdu: Vector3D

Partial derivative of the position p with respect to u

§dpdv: Vector3D

Partial derivative of the position p with respect to v
