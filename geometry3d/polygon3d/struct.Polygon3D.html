Expand description

A 3-dimensional Polygon which can contain holes.

It is a structure containing several Loop3D. One of them is the outer Loop3D, and there there can be several inner Loop3D.

Fields

outer: Loop3D

The outer Loop3D

inner: Vec<Loop3D>

A vector of inner Loop3D

area: f64

The area of the Polygon3D

normal: Vector3D

The normal of the Polygon3D, following a right-hand convention
